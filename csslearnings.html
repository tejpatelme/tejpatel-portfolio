Top 4 CSS Learnings as a beginner

26th November, 2020

CSS stand for Cascading Style Sheets, and it is used to style our boring looking HTML pages .


When we are starting out with CSS it looks so daunting because there so much to learn and the most simplest of things seem so complex to achieve. Even trying to center elements can get you to scratch your head. The key is to not get overwhelmed and take small steps of understanding it.


I've had some similar experience while I was coding this website of mine. But after using my favourite search engine Google to figure things out I've learnt a lot of things which I think will be very useful for beginners just starting out with CSS.

1. All heading tags (H1 to H6) and "p" tag have default margin values.

All heading tags and paragraph tag have a default margin values. This can be a big problem if you are trying to fine tune your spacing around elements. The trick here is to reset the margins of all heading tags and paragraph tab to 0px in your CSS file so you have complete control over the spacing.

2. The unordered list "ul" and ordered list "ol" tags have a default padding of 40px to the left.

Unordered list is our first choice when we want to create navigation elements in our navigation bar or create links to our social media profiles in our footer. Usually we display the list items inside the unordered list in a line using the display:inline property and centering the elements by setting the text-align property of the "ul" or "ol" tag to center however your elements won't look dead centered because of that 40px left padding. The key here is to set the "ul" or "ol" tags padding to 0px.

I won't recommend doing this globally since that default padding is necessary to create hierarchy when we are using lists normally. Ideally you would target the "ul" or "ol" inside your footer or navigation bar and then set the padding the 0px.

3. Add hover animations to your buttons.

4. Using variables to store values that will be reused.

Just type - - followed by the variable name and then store the color values or any value that you want reuse. You can use the variables by specifying their name inside the var() function. I've used variables for this website to store the colors that I have used on this website.
