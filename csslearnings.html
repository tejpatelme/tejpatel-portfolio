Top CSS Tricks for Beginners

26th November, 2020

CSS stand for Cascading Style Sheets, and it is used to style our boring looking HTML pages .


When we are starting out with CSS it looks so daunting because there so much to learn and most simplest of things seem so complex to achieve. Even trying to center elements can get you to scratch your head. The keep is to not get overwhelmed and take small steps of understanding it.


I've had some similar experience while I was coding this website of mine. But after using my favourite search engine Google to figure things out I've learnt a lot of things which I think will be very useful for beginners just starting out with CSS.

1. All heading tags (H1 to H6) and "p" tag have default margin values.

All heading tags and paragraph tag have a default margin values. This can be a very big problem if you trying to fine tune your spacing around elements. The trick here is to reset the margins of all heading tags and paragraph tab to 0px in your CSS file so you have complete control over the spacing.
